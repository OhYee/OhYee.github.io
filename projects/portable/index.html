Portable Development Environment

U盘版集成开发环境

使用VSCode以及一些配置文件实现在U盘完成开发工作

已确认支持语言: C/C++、Java、Python、Node.js
另外也可配置Git环境,因此也可以用来写Hexo博客

编程学习者可以用该环境更方便地进行代码编写工作(比如在网吧)
非编程开发人员也可以用来随时随地更新自己的Hexo博客

设备要求:

  • U盘一个,空间大于2G
  • Windows 7 以上(不包括)电脑(更低版本无法运行VS Code)
    (如果希望在所有系统下保证开发环境,可以使用Deepin来实现)

实现思路:

  • 通过在运行VS Code前使用命令行语句设置临时环境变量,实现开发环境的构建
  • 通过VS Code的参数来指定插件目录和数据目录到U盘内

因此按照该思路,理论上可以实现绝大多数编程环境的U盘化

具体步骤

  1. 下载VS Code ZIP版

  2. 使用cmd来配置临时环境变量,并且设定VS Code启动目录

    前三行是需要用到的参数

    • VSCodePath 是VSCode的路径
    • default_path是默认的环境变量(如果电脑的环境里和U盘里配置了同样的环境,优先使用U盘环境)
    • PortableENV是U盘环境目录,开发环境在该目录里

    下面需要用到那些环境就保留哪一部分即可
    最后是默认打开Code文件夹

  3. 下载对应的环境安装到电脑,安装完成后把文件复制到PortableENV文件夹内,然后电脑内的可以视需要卸载掉

    • git
    • Python
    • nodejs
    • MinGW
    • cmder

  4. 使用run.bat运行VS Code即可
