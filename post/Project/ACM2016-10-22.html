本题解采用 C++ ,大概思路与 C 一样
cin 对应 scanf
cout 对应 printf

如果还有问题群里问即可,有余力的同学可以看群公告里上一次比赛

A + B Problem

计算 A + B
多组数据,以 EOF 结尾

求绝对值

求一个实数的绝对值
使用 cmath 头文件(math.h)里的 fabs() 函数
输出记得保留小数

成绩转换

针对不同的输入,有不同的输出

水仙花数

先写一个函数判断一个数是不是水仙花数
模拟定义即可

可以先把所有数筛选一遍,打个表,可以节省时间

输出的时候记得输出空格,并且只有两个数字中间有,最后不要有多余的

数据的交换输出

遍历所有数,找到最小数的位置,交换它和第一个

字符串统计

遍历一行字符,统计数字的个数

多项式求和

计算多项式

可以先计算好所有的结果,然后查表输出
需要使用 double

求数列的和

使用 cmath 头文件(math.h)的 sqrt() 函数

折线分割平面

可推出公式：2n2 - n + 1

Choosing Teams

给出 n 个人参加过比赛的次数,需要组参加 k 场比赛的 3 人队伍,每个人最多只能参加 5 次比赛
求能组的最多队伍数量

求大于等于 5-k 的人的数量,然后除以 3 即可
